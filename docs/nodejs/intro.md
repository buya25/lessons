**Estimated time per lesson:** ~20 minutes

**After this track, you will be able to:** build backend applications with JavaScript, handle real-time communication, and understand how Node.js differs from browser JavaScript.

## Prerequisites

Complete the [React](../react/intro) track first (so you understand JavaScript well).

## What you will build

A real-time chat feature — users can send and receive messages live without refreshing the page.
