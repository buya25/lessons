# React

**Estimated time per lesson:** ~20 minutes

**After this track, you will be able to:** build interactive web interfaces with React and connect them to a real backend API.

## Prerequisites

Complete the [Python](../python/intro) track first.

## What you will build

An e-commerce frontend — a website where users can browse products, add them to a cart, and place orders. It connects to the Python API and the shared database.
