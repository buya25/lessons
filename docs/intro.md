# Welcome to Code Lessons

This is a free, beginner-friendly programming course that takes you from zero to building real projects.

## How it works

Every track follows the same pattern:

1. **Learn** — lessons that explain one concept at a time (~20 minutes each)
2. **Build** — a real project at the end of every track
3. **Connect** — projects across tracks share the same database and API, so everything fits together

## Where to start

**Start with [Setup](./setup/intro)** — get your computer ready before anything else.

Then follow this order:

| Track | What you build |
|---|---|
| [Database / SQL](./database/intro) | A full e-commerce database schema |
| [Python](./python/intro) | A REST API connected to that database |
| [React](./react/intro) | A frontend that talks to the Python API |
| [Node.js](./nodejs/intro) | A real-time chat feature |
| [PHP](./php/intro) | A blog backend |
| [Java](./java/intro) | An inventory management system |

## One rule

Do not skip lessons. Every lesson is short (~20 minutes) and each one builds on the last.
If you are stuck, read the **When Things Go Wrong** section at the bottom of each lesson.
