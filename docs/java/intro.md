# Java Track

**Estimated time per lesson:** ~20 minutes  
**After this track, you will be able to:** write object-oriented Java, build a REST API with Spring Boot, connect to MySQL with JPA, and understand Java's strict type system.

---

## Prerequisites

Complete the **[Database / SQL](../database/intro)** track first. Java's connection to databases through JPA (Java Persistence API) will make more sense once you understand what SQL is doing underneath.

---

## What You Will Build

An **inventory management REST API** — track products, stock levels, and suppliers. The API stores data in MySQL (the same database schema from the SQL track) and exposes clean JSON endpoints.
