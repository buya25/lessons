# PHP Track

**Estimated time per lesson:** ~20 minutes  
**After this track, you will be able to:** build a server-side web application with PHP, handle forms safely, connect to MySQL with PDO, and structure a multi-page app.

---

## Prerequisites

Complete the **[Database / SQL](../database/intro)** track first — PHP's primary job is talking to a database, and you need to understand SQL before writing PHP that generates it.

---

## What You Will Build

A **blog backend with user accounts** — users can register, log in, write posts, and leave comments. All data is stored in MySQL using the same database you designed in the SQL track.
