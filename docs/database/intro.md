# Database / SQL

**Estimated time per lesson:** ~20 minutes

**After this track, you will be able to:** design and build a full database schema for a real application, write queries to create, read, update, and delete data, and connect a database to a backend application.

## Why start here?

Every application stores data. Before you can build a backend or a frontend,
you need somewhere to put the data. The database you build in this track is
reused by every other track in this course.

## What you will build

A complete e-commerce database — products, users, orders, and payments.
This same database powers the Python API, the React frontend, and the Node.js features.
